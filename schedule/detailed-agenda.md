## üìù Facilitator Notes

### Preparation Checklist
- [ ] Review client's bobathon-config.yaml
- [ ] Customize labs based on client use cases
- [ ] Test all lab exercises in client environment
- [ ] Prepare backup examples for common issues
- [ ] Set up communication channels (Slack, Teams, etc.)
- [ ] Prepare feedback forms
- [ ] Test screen sharing and demo setup

### Tips for Success
1. **Start with quick wins** - Begin with simple, impressive examples
2. **Encourage questions** - Create a safe learning environment
3. **Be flexible** - Adjust pace based on participant needs
4. **Use real examples** - Relate everything to client's actual work
5. **Celebrate progress** - Acknowledge achievements throughout the day
6. **Provide context** - Explain the "why" behind best practices
7. **Follow up** - Schedule office hours and check-ins

### Common Challenges
- **Varying skill levels** - Pair experienced with less experienced participants
- **Technical issues** - Have backup plans and troubleshooting guides ready
- **Time management** - Use timers and be prepared to adjust on the fly
- **Engagement** - Use interactive elements and frequent breaks
- **Complex use cases** - Break down into smaller, manageable pieces
